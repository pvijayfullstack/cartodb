Get user tables

Returns a list of the tables from the user and the metainformation of each of them:

  • name
  • tags
  • privacity
  • schema

URL

/v1/tables

METHOD

GET

EXAMPLE


      GET
        /v1/tables

JSON

      [
        {
          "id": "1",
          "name": "species_occurrences",
          "tags": "species,biodiversity",
          "privacy": "PUBLIC",
          "schema": [["cartodb_id","number"],["name","string"],["description","string"],["latitude","number"],["longitude","number"],["created_at","date"], ["updated_at","date"]]
        },
        {
          "id": "2",
          "name": "free_wifi_hotspots",
          "tags": "",
          "privacy": "PRIVATE",
          "schema": [["cartodb_id","number"],["name","string"],["address","string"],["the_geom","geometry","point"],["created_at","date"], ["updated_at","date"]]
        }
