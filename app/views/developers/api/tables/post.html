Create a table

Creates a new table. Tables can be created importing the schema and the data from a file or empty of data.

In the case of importing a file, it's necessary to provide a valid SRID in which the geometry is supposed to be.

You are able to indicate the type of geometry that your table is going to store. Allowed values are the ones from PostGIS.

URL

/v1/tables

REQUEST METHOD

POST

PARAMETERS

Optional

  • name: the name of the table
  • file: a file which from import data. Accepted formats are: CSV, SHP in a zipfile and XLS(X)
  • the_geom_type: the geometry type of the_geom column.
  • srid: the SRID in which the data from the file is supposed to be

EXAMPLE


      POST
        /v1/tables
          { "name": "species occurrences", :the_geom_type => "Polygon" }

JSON

      {
        "id": "1",
        "name": "species_occurrences"
      }
