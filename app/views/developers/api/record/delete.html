Delete a row

Deletes a row

URL

/v1/tables/<identifier>/records/<primary_key>

METHOD

DELETE

PARAMETERS

Required

  • identifier: the name of the table (in the URL of the request)
  • primary_key: the primary key of the row (in the URL of the request)

EXAMPLE


      DELETE
        /v1/tables/species_occurrences/records/1

The method doesn't return nothing. If you want to check the success, check the HTTP status header.
