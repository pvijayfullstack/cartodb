Create a new row

Creates a new row in the table. Returns the primary key of that row.

If you want to set the geometry field of the row you have to use the_geom and conver the geometry to a GeoJSON representation.

URL

/v1/tables/<identifier>/records

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table (in the URL of the request)
  • The list of colums names and their values

EXAMPLE


      POST
        /v1/tables/species_occurrences/records
          {
            "name": "Laetmonice producta",
            "description": "",
            "the_geom": "{\"type\":\"Point\","coordinates\":[1,2]}",
            "..."
          }

JSON

      {
        "id": 33    
      }
