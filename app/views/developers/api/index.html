CartoDB API

CartoDB offers a REST API that allows you to interact with your tables, fetch rows and perform common database operations + GIS operations in a easy way.

Overwiew

As a CartoDB user, you can create tables to store your data. Every table, which can be public or private, is composed by a set of rows and a set of columns and is identified by a name, which is unique in your database.

Every column has a name, which identifies it and a type. The available types are:

  • String
  • Number
  • Date
  • Boolean

Every row is identified by our primary key, which is a column named cartodb_id.

You need to be authenticated when performing API requests. Refer to the Authentication section to get more information.

Endpoint

The endpoint is http://api.cartodb.com/. You should perform all the request to this host.
