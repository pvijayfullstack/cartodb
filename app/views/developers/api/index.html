CartoDB API

CartoDB is a geospatial database on the cloud that helps you create location aware applications quickly and easily.

Overview

Everything that is done through our user interface can be performed through the API. We have developed 3 separate APIs for different use cases, keep updated with the documentation and don't forget to subscribe to the mailing list to keep updated.

  • REST API
    Allows you to create tables, insert data, read metadata, etc, all in friendly REST way.

  • SQL API
    For advance users, there is nothing better than straight SQL access to their CartoDB.

  • Maps API (pending)
    If you want to embed your maps on other websites.

Authorization

For most operations you will need to authorize with CartoDB before. We have implemented OAuth 1.0 and JSON Token authorization. Read more about Authorization.

Endpoint

The endpoint is https://api.cartodb.com/. You should perform all the request to this host.

The whole API works with JSON. If you need to use JSONP add the parameter "callback".

Libraries

We are working on different libraries for different languages. We have a gem for Ruby and a PHP library. Soon we will have other languages available, but in general all you need is an OAuth client and a JSON parser.

Soon you will also have a Javascript library available to embed CartoDB maps on other websites.
