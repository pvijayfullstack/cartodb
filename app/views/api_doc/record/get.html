Get a single row

Returns the values from a single row

URL

/v1/tables/<identifier>/records/<primary_key>

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table
  • primary_key: the primary key of the row

EXAMPLE


  GET
    /v1/tables/species_occurrences/records/1

JSON

  {
    "cartodb_id": 1,
    "name": "Laetmonice producta",
    "description": "",
    ...
  }
