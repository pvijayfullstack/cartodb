Get the value from a row and a column

Returns the value of a given column in a given row

URL

/v1/table/<identifier>/records/<primary_key>/columns/>column_name<

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table
  • primary_key: the primary key of the row
  • column_name: the name of the column

EXAMPLE


  GET
    /v1/tables/species_occurrences/records/1/columns/name

JSON

  {
    "name": "Laetmonice producta"
  }
