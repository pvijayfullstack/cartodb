Delete a column

Drops a column from the table

URL

/v1/tables/<identifier>/columns/<column_name>

METHOD

DELETE

PARAMETERS

Required

  • identifier: the name of the table (in the URL of the request)
  • column_name: the name of the column (in the URL of the request)

EXAMPLE


  DELETE
    /v1/tables/species_occurrences/columns/name

The method doesn't return nothing. If you want to check the success, check the HTTP status header.
