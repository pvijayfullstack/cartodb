Update the type or the name of a column

Updates the type or the name of a column. When updating the name of the column a new name has to be provided, and when updating the type of the column there are two possibilities:

  • if the data fits, for example, moving from number to string all the values will be kept
  • if the data don't fit, for example, moving from string to date, you may lost the values from the the modified column

So, be careful when using this method.

URL

/v1/tables/<identifier>/columns/<column_name>

METHOD

PUT

PARAMETERS

Required

  • identifier: the name of the table
  • column_name: the name of the column

Optional

  • type: the new type of the column
  • new_name: the new name of the column

EXAMPLE


  PUT
    /v1/tables/species_occurrences/columns/name
      { "type": "number" }

JSON

  { 
    "name": "name", 
    "type": "number" 
  }
