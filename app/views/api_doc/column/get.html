Get the type of a column

Returns the type of a column

URL

/v1/table/<identifier>/columns/<column_name>

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table
  • column_name: the name of the column

EXAMPLE


  GET
    /v1/tables/species_occurrences/columns/name

JSON

  {
    "type": "string"
  }
