Get table information

Returns the metainformation of a table from the user:

  • name
  • tags
  • privacity
  • schema

URL

/v1/table/<identifier>

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table

EXAMPLE


  GET
    /v1/tables/species_occurrences

JSON

  {
    "id": "1",
    "name": "species_occurrences",
    "tags": "species,biodiversity",
    "privacy": "PUBLIC",
    "schema": [["cartodb_id","number"],["name","string"],["description","string"],["latitude","number"],["longitude","number"],["created_at","date"], ["updated_at","date"]]
  }
