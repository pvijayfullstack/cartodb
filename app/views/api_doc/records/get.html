Get the rows

Returns a collection of rows from the table. This collection is limited to 5,000 entries.

URL

/v1/tables/<identifier>/records

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table (in the URL of the request)

Optional

  • page: page number, starting in 0. By default is 0
  • rows_per_page: the number of rows in each page. By default 10

EXAMPLE


  GET
    /v1/tables/species_occurrences/records

JSON

  {
    "id": 1,
    "name": "species_occurrences",
    "total_rows": 10,
    "rows": [
      {
        "cartodb_id": 1,
        "name": "Laetmonice producta",
        "description": "",
        ...
      },
      ...
    ]
  }
