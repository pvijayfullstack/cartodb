Create a table

Creates a new table

URL

/v1/tables

REQUEST METHOD

POST

PARAMETERS

Optional

  • name: the name of the table
  • schema the schema, in SQL, of the table

EXAMPLE


  POST
    /v1/tables
      { "name": "species occurrences" }

JSON

  {
    "id": "1",
    "name": "species_occurrences"
  }
