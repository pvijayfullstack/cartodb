Get the schema from a table

Returns the schema from a table

URL

/v1/tables/<identifier>/columns

METHOD

GET

PARAMETERS

Required

  • identifier: the name of the table (in the URL of the request)

EXAMPLE


  GET
    /v1/tables/species_occurrences/columns

JSON

  [
    ["cartodb_id","number"],["name","string"],
    ["latitude","number","latitude"],["longitude","number","longitude"],
    ["description","string"],["created_at","date"],
    ["updated_at","date"]
  ]
