Add a new column

Adds a new column to the table

URL

/v1/table/<identifier>/columns

METHOD

POST

PARAMETERS

Required

  • identifier: the name of the table
  • name: the name of the new column
  • type: the type of the column

EXAMPLE


  POST
    /v1/tables/species_occurrences/columns
      { "name": "family", "type": "string"}

JSON

  { 
    "name": "family", 
    "type": "string" 
  }
